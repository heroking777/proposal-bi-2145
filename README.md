# 【未経験可】データ基盤・BI開発ができるエンジニアの募集

仕事の詳細
【 概要 】
データ基盤開発（DWHやBIの構築）を担えるエンジニアを募集しています。
データ基盤開発の直接の経験がなくても、システム開発の経験があれば対応可能です。
尚、顧客折衝なども発生するので、日本語ネイティブで、PMのようなビジネスサイドのロールに興味がある方が推奨です。

【 応募方法 】
以下質問に対する回答を必ず応募時に記載ください。

・データ基盤を開発したことはありますか？ある場合は、どのような技術スタックを使用して開発したかを軽くで良いのでご記載ください。ない場合は、過去に開発したことのあるシステムについて、ご記載ください。

【 契約金額(税抜) 】
時給2000-5000円で経験に応じて相談させてください。
※契約金額（税込）からシステム利用料を差し引いた金額が、ワーカーさまの受取金額となります

【 重視する要件 】
- エンジニアとしてシステム開発をした経験
- PMなど顧客対応も積極的に行える方
- AIをフル活用して実装を進められる方

【 推奨要件 】
- BIツールを作った経験1年以上
- SQL, Pythonなどでデータパイプラインを
